# Digital Ocean Deployment Guide

## Quick Deploy

### Option 1: Automated Script (Recommended)

```bash
# Basic deployment
./scripts/deploy-digitalocean.sh <SERVER_IP> <SSH_USER> <GIT_REPO_URL> [DOMAIN]

# Example:
./scripts/deploy-digitalocean.sh 123.45.67.89 root https://github.com/yourusername/hirebit.git yourdomain.com
```

### Option 2: Manual Step-by-Step

1. **SSH into server:**
   ```bash
   ssh root@YOUR_SERVER_IP
   ```

2. **Install dependencies:**
   ```bash
   apt-get update
   apt-get install -y curl git build-essential nginx certbot python3-certbot-nginx
   curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
   apt-get install -y nodejs
   npm install -g pm2
   ```

3. **Clone repository:**
   ```bash
   mkdir -p /var/www/hirebit
   cd /var/www/hirebit
   git clone YOUR_GIT_REPO_URL .
   ```

4. **Install and build:**
   ```bash
   # Backend
   cd /var/www/hirebit/backend
   npm install --production
   npm run build
   
   # Frontend
   cd /var/www/hirebit/frontend
   npm install --production
   npm run build
   ```

5. **Set up environment variables:**
   ```bash
   # Copy env.example files and edit
   cp backend/.env.example backend/.env
   cp frontend/.env.example frontend/.env.local
   nano backend/.env
   nano frontend/.env.local
   ```

6. **Start with PM2:**
   ```bash
   cd /var/www/hirebit
   pm2 start ecosystem.config.js
   pm2 save
   pm2 startup
   ```

7. **Configure Nginx:**
   ```bash
   # Copy nginx config from scripts/nginx-hirebit.conf
   # Or use the config generated by deploy script
   systemctl reload nginx
   ```

8. **Set up SSL (if you have a domain):**
   ```bash
   certbot --nginx -d yourdomain.com
   ```

## Environment Variables

### Backend (.env)
```bash
DATABASE_URL=postgresql://postgres.[PROJECT_REF]:[PASSWORD]@aws-0-[REGION].pooler.supabase.com:6543/postgres
DB_SSL=true
PORT=3001
NODE_ENV=production
JWT_SECRET=your_secure_secret_here
```

### Frontend (.env.local)
```bash
NEXT_PUBLIC_BACKEND_URL=http://YOUR_SERVER_IP:3001
# Or with domain:
# NEXT_PUBLIC_BACKEND_URL=https://yourdomain.com/api
```

## Useful Commands

### Check service status:
```bash
pm2 status
pm2 logs
pm2 logs hirebit-backend
pm2 logs hirebit-frontend
```

### Restart services:
```bash
pm2 restart all
pm2 restart hirebit-backend
pm2 restart hirebit-frontend
```

### Update and redeploy:
```bash
cd /var/www/hirebit
git pull
cd backend && npm install --production && npm run build
cd ../frontend && npm install --production && npm run build
pm2 restart all
```

### Check Nginx:
```bash
nginx -t
systemctl status nginx
systemctl reload nginx
```

## Troubleshooting

### Services not starting:
- Check logs: `pm2 logs`
- Verify environment variables are set
- Check ports are available: `netstat -tulpn | grep -E '3000|3001'`

### Nginx errors:
- Test config: `nginx -t`
- Check error log: `tail -f /var/log/nginx/error.log`

### Database connection issues:
- Verify DATABASE_URL is correct
- Check Supabase firewall allows your server IP
- Test connection: `psql "$DATABASE_URL" -c "SELECT 1"`

## Security Checklist

- [ ] Change default SSH port (optional but recommended)
- [ ] Set up firewall (ufw): `ufw allow 22 && ufw allow 80 && ufw allow 443 && ufw enable`
- [ ] Use strong JWT_SECRET
- [ ] Use strong NEXTAUTH_SECRET
- [ ] Enable SSL/HTTPS
- [ ] Keep system updated: `apt-get update && apt-get upgrade`
